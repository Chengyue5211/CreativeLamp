# 绘创前程 — 儿童原创视觉表达成长平台

## 项目简介
以"原型组合创意绘画"与"图形变形与结构生成"为双训练核心，采用"软件示范→纸上创作→拍照上传→AI评估→展示归档→家长管理→专属文创"闭环的儿童原创视觉表达平台。

## 教育原则（创始人原创理论）
- **核心公式**：没有原型，就没有变形；没有变形，就没有创造
- **双训练体系**：模块A（原型组合创意绘画）+ 模块B（图形变形与结构生成）
- **108种线条原型**：9大类开放性线条 × 12种变式
- **30种变形方法**：7大母类（尺度形态/结构/空间/状态/关系/创意变形）
- **九级年龄梯度**：准备级(3-4岁) → 超级(初中+)
- **纸屏协同**：软件示范+出题，孩子在纸上真实创作，拍照上传

## 技术栈
- 后端：Python 3.11+ / FastAPI / Uvicorn
- 数据库：SQLite (主库) + ChromaDB (向量库，可选)
- AI：阿里通义千问 (主) + Claude (备)
- 前端：PWA Web应用，移动端优先，HTML+CSS+JS
- 存证：蚂蚁链（复用年轮项目能力）

## 六大产品系统
1. **双训练系统** — 模块A(原型组合) + 模块B(图形变形)
2. **作品系统** — 拍照上传、AI识别、结构化入库、元数据管理
3. **展示系统** — 个人展馆、家庭共享区、公共展示区、成长档案
4. **家长系统** — 作品路径管理、成长快报、候选升级决策
5. **候选升级系统** — 文创候选、存证候选、确权候选
6. **分享推广系统** — 双边激励、三段式奖励、裂变传播

## 开发规范
- 每次修改完成后：commit → push
- 追求完整性和品质，不走捷径
- 以做自己产品的精神，主动推进
- API优先设计，前后端分离
- 所有用户输入必须验证和转义
- 数据库使用上下文管理器，不裸连接

## 目标用户
- 主要用户：4-12岁儿童
- 决策用户：家长（作品路径管理者）
- 扩展用户：老师、教育机构
