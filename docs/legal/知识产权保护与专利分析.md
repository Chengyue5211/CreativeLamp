### 3.2 基于九级体系的个性化训练任务生成方法

| 项目 | 内容 |
|------|------|
| **专利名称** | 一种基于九级进阶体系的儿童视觉表达个性化训练任务自动生成方法 |
| **专利类型** | 发明专利 |
| **技术领域** | 计算机辅助教育、自适应学习系统 |
| **技术问题** | 现有儿童创造力训练软件多采用固定题库或简单随机出题方式，无法根据儿童的年龄、能力水平和已完成训练情况进行精准的个性化任务匹配，导致训练难度与儿童实际能力脱节。 |
| **技术方案** | 设计一种包含九个递进等级的个性化任务生成系统：（1）**等级体系设计**：将儿童视觉表达能力划分为九个等级——准备级（3-4岁）、初级上（4-5岁）、初级下（5-6岁）、中级上（6-8岁）、中级下（8-10岁）、高级上（10-12岁）、高级下（12岁以上）、超级上（初中）、超级下（初中以上），每个等级对应特定的能力参数配置；（2）**参数化任务配置**：每个等级分别配置模块A参数（最大原型数量1-8、可用构图方式集合、可用任务类型集合）和模块B参数（最大图形数量1-5、最大动作数量1-7、难度等级1-5、可用任务类型集合）；（3）**受控随机任务生成**：根据儿童当前等级的参数约束，从原型库中随机抽取指定数量的原型，从动作库中随机抽取指定数量的变形动作，随机选择一种构图方式，并为轮廓装饰类任务基于难度匹配推荐合适的轮廓图形，最终组合生成结构化训练任务；（4）**每日双任务生成**：每天自动为每个儿童生成一个模块A任务和一个模块B任务。 |
| **有益效果** | （1）九级体系覆盖3岁至初中以上年龄段，实现全年龄段精准适配；（2）参数化配置使任务难度严格可控，避免过难或过易；（3）受控随机机制在保证教学有效性的同时提供足够的任务多样性；（4）双任务模式同时训练组合能力与生成能力。 |
| **权利要求要点** | 1. 一种儿童视觉表达个性化训练任务自动生成方法，其特征在于：根据儿童所处的能力等级，从预设的九级参数配置中获取对应等级的约束参数……<br>2. 根据权利要求1所述方法，其中所述能力等级包括九个递进等级…… |

### 3.3 双训练模块协同的儿童创造力培养系统

| 项目 | 内容 |
|------|------|
| **专利名称** | 一种基于原型组合与图形变形双模块协同的儿童创造力培养系统 |
| **专利类型** | 发明专利 |
| **技术领域** | 计算机辅助教育系统、儿童智力开发 |
| **技术问题** | 现有儿童绘画训练系统通常只提供单一训练维度，要么侧重图形组合，要么侧重造型变化，无法同时系统性地培养儿童的"组合能力"与"生成能力"两种底层创造力。 |
| **技术方案** | 构建一种双训练模块协同工作的儿童创造力培养系统，包括：（1）**模块A——原型组合创意绘画**：基于108种线条原型和9种构图方式，训练儿童将简单原型组织成有节奏、有结构、有作品感的画面的能力，任务类型包括识别、模仿、补全、迁移、创作、轮廓装饰和扩图7种；（2）**模块B——图形变形与结构生成**：基于9种基础图形和7种变形动作（含增加动作的5种子类型），训练儿童从已有形态变出新形态的能力，任务类型包括单形变形、双形拼接、多形结构生成、主题生成、系列生成、增加细节和增加附属7种；（3）**统一平台层**：两个模块共享统一的账户系统、任务分发引擎、纸屏协同创作流程、作品上传与管理系统、四维度评价体系、展示系统、成长档案系统和推广裂变系统；（4）**协同机制**：每日为每个儿童同时生成一个模块A任务和一个模块B任务，两个模块的作品进入同一个作品系统和成长档案，形成组合能力与生成能力的交叉训练效果。 |
| **有益效果** | （1）双模块协同设计填补了单一维度训练的不足，实现"组合"与"生成"两种创造力的同步培养；（2）统一平台层设计降低了系统复杂度，使两个模块能够无缝协同；（3）模块化架构支持未来扩展更多训练模块而无需重构平台；（4）跨模块的统一作品系统和成长档案能够全面反映儿童的创造力发展状况。 |
| **权利要求要点** | 1. 一种双模块协同的儿童创造力培养系统，其特征在于包括：原型组合创意绘画训练模块、图形变形与结构生成训练模块，以及统一的平台服务层……<br>2. 根据权利要求1所述系统，其中所述原型组合训练模块包含108种线条原型和9种构图组合方式……<br>3. 根据权利要求1所述系统，其中所述图形变形模块包含9种基础图形和7种变形动作…… |

### 3.5 儿童作品四维度评价与成长追踪方法

| 项目 | 内容 |
|------|------|
| **专利名称** | 一种基于四维度量化评分的儿童视觉表达作品评价与成长追踪方法 |
| **专利类型** | 发明专利 |
| **技术领域** | 教育评价技术、数据分析 |
| **技术问题** | 现有儿童绘画评价多为主观定性评价，缺少结构化的量化维度；同时，评价结果通常是孤立的，无法形成连续的成长轨迹供纵向比较和分析。 |
| **技术方案** | 提出一种四维度量化评价与成长追踪方法：（1）**四维度评价模型**：每件作品从原创性（originality，0-10分）、细节丰富度（detail，0-10分）、构图表现（composition，0-10分）、创意表达（expression，0-10分）四个维度进行量化评分，计算综合均分；（2）**多来源评价**：支持儿童自评和家长评价两种评价来源，评价结果关联至具体作品与训练任务；（3）**成长档案系统**：将评价结果按时间、模块、等级、系列等多维度归档，形成连续的成长轨迹；（4）**可视化成长展示**：在个人展馆中呈现四维度的平均分趋势、各维度进步曲线、优秀作品精选集，支持阶段性成长回顾；（5）**评价-训练联动**：评价结果自动关联至对应训练任务，更新任务状态为已评价，形成"训练-上传-评价-成长记录"的完整闭环。 |
| **有益效果** | （1）四维度量化评价使儿童创作能力的衡量从主观印象转向结构化数据；（2）纵向成长追踪使家长和教师能够清晰看到儿童在各维度的进步轨迹；（3）多来源评价丰富了评价视角；（4）评价-训练联动确保每次训练都有可追踪的成长记录。 |
| **权利要求要点** | 1. 一种儿童视觉表达作品四维度评价方法，其特征在于：对每件作品从原创性、细节丰富度、构图表现、创意表达四个维度分别进行0-10分的量化评分……<br>2. 根据权利要求1所述方法，还包括将评价结果按时间序列归档形成成长轨迹的步骤…… |
| **申请优先级** | **中** — 评价体系是平台的重要差异化特征，但独立专利的保护范围相对较窄 |
