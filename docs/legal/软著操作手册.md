# 绘创前程 软件操作手册

## 软件名称：绘创前程

## 版本号：V1.0

---

## 一、软件概述

### 1.1 产品定位

绘创前程是一款面向4-12岁儿童的原创视觉表达成长平台。平台以"原型组合创意绘画"与"图形变形与结构生成"为双训练核心，采用"软件示范、纸上创作、拍照上传、AI评估、展示归档、家长管理、专属文创"的完整闭环体系，帮助儿童系统性地培养视觉表达与创造力。

### 1.2 核心教育理念

本软件基于创始人原创教育理论，核心公式为："没有原型，就没有变形；没有变形，就没有创造"。通过108种线条原型、9种基础图形、7种变形方法（含30种子方法）以及44种轮廓图形的系统化训练，引导儿童从模仿走向创造。

### 1.3 系统架构

本软件采用B/S（Browser/Server）架构，基于Python FastAPI框架构建后端服务，前端采用HTML5/CSS3/JavaScript原生技术栈开发，支持PWA（渐进式Web应用）模式。用户通过现代浏览器即可访问使用，无需安装额外客户端软件。

### 1.4 主要功能模块

本软件包含以下六大核心系统：

1. **双训练系统**：模块A（原型组合创意绘画）+ 模块B（图形变形与结构生成），提供每日训练任务生成、原型库浏览、变形动作库浏览、轮廓图形库浏览、训练底稿打印等功能。
2. **作品系统**：支持拍照/选图上传、AI四维度评价（原创性/细节/构图/表达力）、作品可见性管理、候选状态管理等功能。
3. **展示系统**：包含个人展馆、家庭共享区、公共展示区和成长档案功能。
4. **家长系统**：提供家长仪表盘、孩子作品管理、家长评价等家长端管理功能。
5. **推广系统**：包含邀请码生成、分享邀请链接、推广数据统计、学币余额管理、里程碑奖励等裂变推广功能。
6. **文创商城**：提供基于孩子原创作品的文创产品浏览、学币抵扣下单、订单管理等商城功能。

## 二、运行环境

### 2.1 服务端环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11、macOS 12+、Ubuntu 20.04+ 或其他主流Linux发行版 |
| 运行时 | Python 3.10 及以上版本 |
| Web框架 | FastAPI + Uvicorn |
| 数据库 | SQLite 3.35+ |
| 内存 | 建议 2GB 以上 |
| 硬盘空间 | 建议 10GB 以上（含用户上传图片存储） |
| 网络 | 需要互联网连接（用于AI评价服务调用） |

### 2.2 客户端环境要求

| 项目 | 要求 |
|------|------|
| 浏览器 | Google Chrome 90+、Microsoft Edge 90+、Apple Safari 14+、Mozilla Firefox 88+ |
| 屏幕分辨率 | 建议 375px 宽度以上（移动端优先设计） |
| 网络 | 需要互联网连接 |

### 2.3 支持平台

- **桌面端**：Windows、macOS、Linux（通过浏览器访问）
- **移动端**：iOS、Android（通过浏览器访问或添加至主屏幕使用PWA模式）

### 2.4 软件安装与启动

#### 步骤一：安装Python依赖

在服务端命令行中执行以下命令安装项目所需依赖包：

```
pip install -r requirements.txt
```

#### 步骤二：启动服务

在项目根目录下执行以下命令启动后端服务：

```
python main.py
```

服务默认监听8102端口。

#### 步骤三：访问系统

在浏览器地址栏中输入服务地址（如 `http://localhost:8102`）即可打开系统首页。

## 三、功能操作说明

### 3.1 家长注册与登录

#### 3.1.1 手机号注册

1. 打开软件首页，点击页面底部的【注册】按钮或链接，进入注册页面。
2. 在"手机号"输入框中输入家长本人的手机号码（11位中国大陆手机号）。
3. 在"密码"输入框中设置登录密码，密码长度不少于6位。
4. 在"确认密码"输入框中再次输入密码，确保两次输入一致。
5. （可选）如果持有邀请码，在"邀请码"输入框中填写邀请码。通过邀请码注册可获得额外学币奖励。
6. 点击【注册】按钮提交注册信息。
7. 系统验证信息无误后，自动完成注册并跳转至登录页面，提示"注册成功"。

**注意事项**：
- 每个手机号仅能注册一个账号。
- 邀请码为选填项，填写有效邀请码后，邀请人和被邀请人均可获得学币奖励。

#### 3.1.2 登录

1. 打开软件首页，进入登录页面。
2. 在"手机号"输入框中输入已注册的手机号码。
3. 在"密码"输入框中输入登录密码。
4. 点击【登录】按钮。
5. 系统验证账号密码正确后，自动登录并跳转至家长仪表盘主页。

#### 3.1.3 登出

1. 在任意页面，点击页面右上角或底部导航栏中的【我的】/【设置】入口。
2. 在个人中心页面，点击【退出登录】按钮。
3. 系统清除当前登录状态，返回登录页面。

### 3.2 孩子管理

#### 3.2.1 添加孩子

1. 登录后进入家长仪表盘，点击【添加孩子】按钮。
2. 在弹出的表单中填写以下信息：
   - **昵称**：输入孩子的昵称（必填，用于系统内显示）。
   - **年龄**：选择孩子的实际年龄（必填，用于自动分级）。
   - **性别**：选择孩子的性别（男/女）。
3. 点击【确认添加】按钮，系统保存孩子信息。
4. 系统根据孩子年龄自动分配对应的训练级别。

**九级年龄梯度体系**：

| 级别 | 名称 | 适用年龄 |
|------|------|----------|
| 第一级 | 准备级 | 3-4岁 |
| 第二级 | 启蒙级 | 4-5岁 |
| 第三级 | 基础级 | 5-6岁 |
| 第四级 | 进阶级 | 6-7岁 |
| 第五级 | 发展级 | 7-8岁 |
| 第六级 | 提高级 | 8-9岁 |
| 第七级 | 拓展级 | 9-10岁 |
| 第八级 | 创作级 | 10-12岁 |
| 第九级 | 超级 | 初中及以上 |

#### 3.2.2 查看孩子信息

1. 在家长仪表盘页面，可查看已添加的所有孩子列表。
2. 每个孩子卡片上显示昵称、年龄、当前级别、作品数量等基本信息。
3. 点击孩子卡片可进入该孩子的详细信息页面。

#### 3.2.3 切换孩子模式

1. 当家长账号下绑定了多个孩子时，在家长仪表盘或训练页面的顶部会显示当前选中的孩子。
2. 点击孩子名称区域或【切换孩子】按钮，弹出孩子选择列表。
3. 在列表中点击需要切换到的孩子，系统自动刷新页面内容，展示该孩子对应的训练任务、作品等数据。

### 3.3 训练系统

训练系统是本软件的核心功能模块，包含模块A（原型组合创意绘画）和模块B（图形变形与结构生成）两大训练体系。

#### 3.3.1 生成每日训练任务

1. 在家长仪表盘或底部导航栏中点击【训练】入口，进入训练系统主页。
2. 系统显示当前选中孩子的今日训练任务。若当日尚未生成任务，页面提示"暂无今日任务"。
3. 点击【生成今日任务】按钮，系统根据孩子的当前级别和训练进度，自动生成一组适合该级别的训练任务。
4. 生成完成后，页面刷新显示今日训练任务列表，包含任务名称、训练类型（模块A/模块B）、难度等级等信息。

**说明**：
- 每日训练任务根据孩子的级别自动适配难度。
- 任务内容由系统AI智能生成，确保每日训练内容的新鲜感和递进性。
- 已完成的历史任务可在"任务历史"中查看。

#### 3.3.2 查看任务详情

1. 在训练任务列表中，点击某一具体任务卡片，进入任务详情页面。
2. 任务详情页面展示以下信息：
   - **任务名称**：本次训练的主题名称。
   - **训练类型**：所属模块（模块A或模块B）。
   - **训练目标**：本次训练的具体目标描述。
   - **使用原型/图形**：本次训练涉及的线条原型、基础图形或轮廓。
   - **变形方法**：本次训练采用的变形方法。
   - **示范说明**：图文形式的训练示范和步骤指引。
   - **难度等级**：标注本次任务的难度级别。
3. 家长可根据任务详情指导孩子在纸上进行创作。

#### 3.3.3 原型库浏览

1. 在训练系统主页或任务详情页中，点击【原型库】入口，进入原型库浏览页面。
2. 原型库包含两大分类：

**（一）108种线条原型**

系统将108种线条原型分为9大类别，每类包含12种变式：
- 直线类、曲线类、折线类、弧线类、波浪线类、螺旋线类、锯齿线类、点线类、自由线类。
- 用户可点击任一类别名称展开查看该类别下的12种变式。
- 每种线条原型以图示方式展示其形态特征和绘制方法。

**（二）9种基础图形**

- 包含圆形、三角形、正方形、长方形、椭圆形、菱形、梯形、五角星、心形等9种基础图形。
- 每种基础图形展示标准图示和变化形态。

3. 用户可通过页面顶部的分类标签快速切换浏览不同类别。
4. 点击任一原型可查看大图和详细说明。

#### 3.3.4 变形动作库浏览

1. 在训练系统主页或任务详情页中，点击【变形动作库】入口，进入变形方法浏览页面。
2. 系统展示7种变形母类，每种母类下包含若干子方法，共计30种变形方法：

| 母类 | 说明 |
|------|------|
| 尺度变形 | 大小缩放、长短变化、粗细调整等 |
| 形态变形 | 弯曲、扭转、拉伸、压缩等形态改变 |
| 结构变形 | 分割、组合、嵌套、叠加等结构重组 |
| 空间变形 | 旋转、翻转、平移、对称等空间变换 |
| 状态变形 | 动静转换、虚实变化、渐变过渡等 |
| 关系变形 | 穿插、包围、并列、递进等关系处理 |
| 创意变形 | 拟人、拟物、联想、夸张等创意手法 |

3. 点击任一变形母类可展开查看该类别下的所有子方法。
4. 每种变形方法配有图文示例，直观展示变形前后的对比效果。

#### 3.3.5 轮廓图形库浏览

1. 在训练系统主页或任务详情页中，点击【轮廓图形库】入口，进入轮廓库浏览页面。
2. 系统展示44种轮廓图形，涵盖动物、植物、器物、建筑、交通工具等多种分类。
3. 每种轮廓图形以简笔轮廓线方式呈现，用户可浏览其标准形态。
4. 点击任一轮廓可查看大图、轮廓名称及分类信息。
5. 轮廓图形主要用于模块B（图形变形与结构生成）的训练任务。

#### 3.3.6 打印训练底稿

1. 在任务详情页面中，点击【打印底稿】按钮。
2. 系统生成该任务对应的可打印训练底稿页面，底稿内容包括：
   - 任务名称和训练目标。
   - 参考原型/图形的图示。
   - 创作区域留白（供孩子在纸上创作）。
   - 任务编号和日期标注。
3. 浏览器弹出打印预览窗口，用户可选择打印机并调整打印设置。
4. 点击【打印】完成底稿打印。
5. 家长将打印好的底稿交给孩子，孩子在纸上完成创作。

### 3.4 作品系统

#### 3.4.1 拍照/选择图片上传作品

1. 在训练任务详情页面中，完成纸上创作后，点击【上传作品】按钮。
2. 系统弹出上传方式选择：
   - **拍照上传**：调用设备摄像头，对准孩子的纸上作品拍照。拍照完成后预览照片，确认清晰无误后点击【确认】。
   - **从相册选择**：打开设备相册，选择之前拍摄好的作品照片。
3. 照片选定后，系统显示上传预览，用户可对照片进行以下操作：
   - 确认照片清晰度和完整度。
   - 如需重新拍摄或选择，点击【重新选择】。
4. 点击【提交上传】按钮，系统开始上传作品图片。
5. 上传成功后，系统自动进行AI评价处理（详见3.4.2），并在作品列表中显示新上传的作品。

**注意事项**：
- 支持的图片格式包括JPEG、PNG。
- 单张图片大小限制为10MB。
- 建议在光线充足的环境下拍照，确保作品清晰可辨。

#### 3.4.2 作品评价

作品上传后，系统自动调用AI进行四维度综合评价：

| 评价维度 | 说明 |
|----------|------|
| 原创性 | 评估作品的创意独特程度和原创表达 |
| 细节 | 评估作品的细节丰富程度和精细表现 |
| 构图 | 评估作品的画面布局和空间组织能力 |
| 表达力 | 评估作品的情感表达和视觉传达效果 |

1. AI评价完成后，系统在作品详情页面展示各维度的评分（分值范围根据级别设定）。
2. 同时生成文字评语，对孩子的创作给出鼓励性的反馈和建设性的建议。
3. 评价结果自动记入孩子的成长档案，作为能力发展数据的一部分。

#### 3.4.3 作品详情查看

1. 在作品列表页面（可通过底部导航栏的【作品】入口进入），点击任一作品卡片。
2. 进入作品详情页面，展示以下信息：
   - **作品图片**：高清大图展示。
   - **作品名称**：系统自动生成或用户自定义的名称。
   - **创作日期**：作品上传的日期。
   - **关联任务**：该作品对应的训练任务信息。
   - **AI评价**：四维度评分和文字评语。
   - **家长评价**：家长对作品的文字点评（如已添加）。
   - **可见性状态**：当前作品的可见范围（私密/家庭/公开）。
   - **候选状态**：当前作品的候选标记（文创/存证/确权）。

#### 3.4.4 作品可见性管理

1. 在作品详情页面中，找到"可见性"设置区域。
2. 系统提供三种可见性级别：
   - **私密**：仅家长和孩子本人可见，不对外展示。
   - **家庭**：家庭成员可见，可在家庭共享区查看。
   - **公开**：所有用户可见，可在公共展示区展示。
3. 点击对应的可见性选项进行切换。
4. 系统立即更新作品的可见范围，生效后在对应展示区域显示或隐藏。

**默认设置**：新上传的作品默认为"私密"状态，需家长主动调整可见性。

#### 3.4.5 候选状态管理

1. 在作品详情页面中，找到"候选状态"管理区域。
2. 系统提供三种候选标记：
   - **文创候选**：将作品标记为文创产品制作候选，后续可用于生成个性化文创商品。
   - **存证候选**：将作品标记为区块链存证候选，用于作品原创性存证保护。
   - **确权候选**：将作品标记为版权确权候选，用于作品版权登记。
3. 点击对应的候选标记按钮，切换作品的候选状态（选中/取消）。
4. 一件作品可同时拥有多个候选标记。
5. 被标记为候选的作品将出现在家长中心的对应候选列表中，便于后续统一处理。

### 3.5 展示与成长

#### 3.5.1 作品展馆

1. 在底部导航栏中点击【展馆】入口，进入作品展馆页面。
2. 作品展馆包含以下展示区域：
   - **个人展馆**：展示当前孩子的所有公开作品，以瀑布流或网格形式呈现。
   - **家庭共享区**：展示家庭内所有孩子设为"家庭"可见的作品。
   - **公共展示区**：展示平台内所有用户设为"公开"的优秀作品。
3. 用户可在页面顶部通过标签切换不同展示区域。
4. 在公共展示区，作品按上传时间倒序排列，用户可滑动浏览。
5. 点击任一展示作品可查看作品详情和评价信息。

#### 3.5.2 成长档案

1. 在家长仪表盘或底部导航栏中点击【成长档案】入口，进入当前孩子的成长档案页面。
2. 成长档案页面展示以下内容：

**（一）统计数据**

- 累计创作作品数量。
- 累计完成训练任务数量。
- 当前训练级别及晋级进度。
- 连续创作天数记录。

**（二）能力雷达图**

- 以雷达图形式展示孩子在四个评价维度（原创性/细节/构图/表达力）上的综合能力水平。
- 雷达图数据基于所有历史作品的AI评价得分计算平均值。
- 直观呈现孩子的能力强项和待提升方向。

**（三）月度趋势**

- 以折线图形式展示孩子各月度的创作数量变化趋势。
- 以折线图形式展示各评价维度得分的月度变化趋势。
- 用户可切换查看不同月份的数据。
- 帮助家长了解孩子的创作活跃度和能力成长曲线。

#### 3.5.3 任务历史

2. 系统按时间倒序展示所有已生成的训练任务。
3. 每条任务记录显示以下信息：
   - 任务日期。
   - 任务名称和训练类型。
   - 完成状态（已完成/未完成）。
   - 关联作品数量。
4. 点击任一历史任务可查看任务详情及对应的作品。
5. 对于未完成的历史任务，用户仍可上传作品完成任务。

#### 3.6.1 家长仪表盘

1. 登录后默认进入家长仪表盘页面，该页面为家长端的主控制台。
2. 仪表盘页面展示以下核心信息：
   - **孩子概览**：已添加孩子的列表及各孩子的基本状态信息。
   - **今日任务**：当前选中孩子的今日训练任务概要。
   - **最近作品**：最近上传的作品缩略图预览。
   - **成长快报**：孩子近期创作活跃度和能力变化的简要汇报。
   - **学币余额**：当前账户的学币余额显示。

#### 3.6.2 孩子作品管理

1. 在家长仪表盘中点击某个孩子的卡片，或点击【作品管理】入口。

   - **设置候选**：将优秀作品标记为文创/存证/确权候选。
   - **筛选过滤**：按时间、评分、可见性、候选状态等条件筛选作品。
4. 作品列表支持按创作时间排序或按评分排序。

#### 3.6.3 家长评价

1. 在作品详情页面中，找到"家长评价"区域。
2. 点击【添加评价】按钮，弹出评价输入框。
3. 家长在输入框中撰写对孩子作品的文字点评，内容可包括：
   - 对孩子创作过程的鼓励。
   - 对作品亮点的认可。
   - 温和的建议和引导。
4. 点击【提交评价】按钮，家长评价保存并显示在作品详情页面中。
5. 家长评价作为成长档案的一部分，与AI评价共同记录孩子的创作成长历程。
6. 已提交的评价支持修改和删除。

#### 3.7.4 学币余额和流水

## 四、安全机制

### 4.1 身份认证

本软件采用JWT（JSON Web Token）令牌认证机制保障用户身份安全：
- 用户登录成功后，服务端生成加密的JWT令牌并返回客户端。
- 客户端在后续每次请求中携带该令牌作为身份凭证。
- 令牌设有有效期，过期后需重新登录。
- 服务端对每次请求进行令牌校验，确保请求来自合法用户。

### 4.2 密码安全

- 用户密码使用bcrypt算法进行加密存储，不以明文形式保存。
- bcrypt算法具有自适应哈希特性，可有效抵御暴力破解攻击。
- 数据库中仅存储密码的哈希值，即使数据库泄露也无法直接获取用户密码。

### 4.3 CSRF防护

- 系统实施CSRF（跨站请求伪造）防护机制。
- 对所有状态变更请求（如提交表单、修改数据）进行CSRF令牌验证。
- 防止恶意网站冒用用户身份发起非法请求。

### 4.4 XSS防护

- 系统对所有用户输入内容进行严格的转义和过滤处理。
- 防止XSS（跨站脚本攻击）恶意代码注入。
- 输出内容均经过HTML实体编码，确保用户数据安全展示。

### 4.5 速率限制

- 系统对API接口实施速率限制策略。
- 限制同一IP或同一用户在单位时间内的请求频率。
- 有效防止暴力破解、恶意刷取等自动化攻击行为。

### 4.6 文件上传安全校验

- 对用户上传的文件进行多重安全校验：
  - **文件类型校验**：仅允许指定的图片格式（JPEG、PNG）。
  - **文件大小限制**：单文件大小不超过10MB。
  - **文件内容检测**：校验文件头信息，防止伪装文件类型的恶意文件上传。
  - **文件名安全处理**：对上传文件名进行安全过滤，防止路径穿越攻击。

## 五、技术支持

### 5.1 常见问题

**问：注册时提示"手机号已存在"？**
答：每个手机号仅能注册一个账号。如忘记密码，请联系技术支持协助重置。

**问：上传作品时提示失败？**
答：请检查以下几点：（1）网络连接是否正常；（2）图片格式是否为JPEG或PNG；（3）图片大小是否超过10MB限制。如问题持续，请尝试更换浏览器或清除浏览器缓存后重试。

**问：AI评价未显示？**
答：AI评价需要调用云端服务，可能因网络延迟导致评价结果稍有延迟。请稍候片刻后刷新页面查看。如长时间未出现评价结果，请联系技术支持。

**问：如何将网页添加至手机主屏幕？**
答：在移动端浏览器中打开本系统后，点击浏览器菜单中的"添加到主屏幕"选项，即可将本软件以PWA模式安装至手机主屏幕，获得类似原生应用的使用体验。
