## 2. 原型库的作用

原型库是模块A“原型组合创意绘画”的内容基础。

它的作用不是单纯存素材，而是为平台提供：

- 原型学习内容
- 原型示范内容
- 任务生成的原型池
- 作品元数据标签
- 搜索与统计依据
- 后续教学和内容扩展的基础字典

没有统一原型库，模块A 就无法稳定地：

- 出任务
- 做标签
- 做评价
- 做成长统计

---

## 3. 原型库的设计原则

### 3.1 原型必须是“可训练单位”
原型不是随便找一些线条图片，而必须是：
- 可识别
- 可命名
- 可重复训练
- 可进入组合任务

### 3.2 原型要优先服务训练，不是优先服务分类学
平台中的原型分类，不必追求学术美术分类的复杂度，而要优先考虑儿童学习与任务生成是否清楚。

### 3.3 原型命名必须稳定
同一个原型，不允许在不同文件、不同页面里换不同叫法。

### 3.4 原型必须支持“平台化使用”
一个原型不仅用于展示，还应支持：
- 编码
- 标签化
- 模板调用
- 元数据记录
- 评价引用

## 4. 模块A 当前建议使用的基础原型

平台首发阶段，建议先统一采用以下 9 个基础线条原型。

### 4.1 点
- 中文名：点
- 英文编码：`dot`
- 特征：独立的点状元素，可重复、可聚集、可疏密变化

### 4.2 直线
- 中文名：直线
- 英文编码：`line_straight`
- 特征：线条笔直，方向感强

### 4.3 虚线
- 中文名：虚线
- 英文编码：`line_dashed`
- 特征：由间断线段构成，适合做轻节奏装饰

### 4.4 弧线
- 中文名：弧线
- 英文编码：`line_arc`
- 特征：柔和弯曲，适合流动型结构

### 4.5 波浪线
- 中文名：波浪线
- 英文编码：`line_wave`
- 特征：连续起伏，适合节奏型画面

### 4.6 折线
- 中文名：折线
- 英文编码：`line_zigzag`
- 特征：有转折、有棱角，适合动感与尖锐感

### 4.7 城墙线
- 中文名：城墙线
- 英文编码：`line_wall`
- 特征：类似台阶或齿状节奏，适合建筑感、结构感

### 4.8 螺旋线
- 中文名：螺旋线
- 英文编码：`line_spiral`
- 特征：旋转向内或向外，适合中心感、卷动感

### 4.9 弹簧线
- 中文名：弹簧线
- 英文编码：`line_spring`
- 特征：回旋连续，富于弹性和趣味性

## 7. 原型命名规则

### 7.1 中文命名
中文名必须统一，不得出现：
- 同一原型多个叫法
- 页面名称和后台名称不一致

例如：
统一写“波浪线”，不要有时写“波纹线”。

### 7.2 英文编码命名
建议采用：
`类别_名称`

例如：
- `dot`
- `line_straight`
- `line_arc`
- `line_wave`
- `line_zigzag`
- `line_wall`
- `line_spiral`
- `line_spring`

### 7.3 文件命名
素材文件建议采用：

`原型序号_英文编码_版本号`

例如：
- `01_dot_v1.png`
- `02_line_straight_v1.png`
- `03_line_dashed_v1.png`
