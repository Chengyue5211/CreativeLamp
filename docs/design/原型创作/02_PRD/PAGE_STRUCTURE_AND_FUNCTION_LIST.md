# 4. 首页

## 4.1 页面定位
首页是整个平台的总入口。

它不只是功能列表，而应该让用户一进来就明白三件事：

- 今天可以做什么训练
- 最近完成了什么作品
- 作品接下来可以去哪里

## 4.2 核心模块

### 模块1：今日任务入口
显示：
- 今日推荐任务
- 当前训练模块
- 当前等级
- 开始训练按钮

### 模块2：最近作品预览
显示最近上传的 1—3 件作品缩略图。

### 模块3：我的展馆入口
进入孩子个人展示区。

### 模块4：成长进度卡
显示：
- 已完成任务数
- 本周作品数
- 当前训练等级
- 最近解锁内容

### 模块5：精选 / 主题展入口
展示：
- 本周精选
- 推荐作品
- 推荐系列

### 模块6：家长操作提醒
如：
- 有作品待确认展示
- 有作品可加入候选
- 有奖励待查看

## 4.3 功能清单
- 查看今日任务
- 进入训练模块
- 查看最近作品
- 进入个人展示区
- 查看成长进度
- 接收家长待办提醒

# 5. 训练中心

## 5.1 页面定位
训练中心是平台最核心的能力入口。

负责承接两个训练模块：

- 原型组合创意绘画
- 图形变形与结构生成

## 5.2 页面结构

### 模块1：训练模块选择区
两个大入口卡片：

#### 卡片A
原型组合创意绘画

#### 卡片B
图形变形与结构生成

每张卡片显示：
- 模块简介
- 当前等级
- 最近完成情况
- 进入按钮

### 模块2：我的训练进度
显示：
- 当前模块完成度
- 已解锁等级
- 最近作品

### 模块3：推荐练习
根据孩子当前阶段推荐 1—3 个任务。

### 模块4：兴趣选图入口（后续）
适合熟练后进入：
- 动物
- 植物
- 物品
- 建筑
- 梦幻角色

### 模块5：扩图创作入口（后续）
面向已熟练孩子的升级入口。

## 5.3 功能清单
- 选择训练模块
- 查看模块说明
- 查看当前等级
- 进入推荐任务
- 进入兴趣选图
- 进入扩图创作
- 查看训练记录

# 8. 学习示范页

## 8.1 页面定位
这一页专门负责“教”，不是让孩子在上面完成作品，而是让孩子看懂方法。

## 8.2 页面结构

### 模块1：动态示范区
演示：
- 原型怎么画
- 图形怎么变
- 怎么分区
- 怎么组合
- 怎么扩图
- 怎么增加细节与附属对象

### 模块2：步骤提示区
用简短步骤提示：
1. 先看  
2. 再想  
3. 再分区 / 变形  
4. 再完成

### 模块3：对比示范区
显示：
- 基础形 / 变化后
- 单一原型 / 组合后
- 单体 / 扩图后

### 模块4：开始任务按钮
看完即可进入任务页。

## 8.3 功能清单
- 观看示范
- 切换步骤
- 查看前后对比
- 进入任务

# 9. 任务页

## 9.1 页面定位
任务页是平台最关键的执行页。

所有训练最终都要落到任务页上。

## 9.2 页面结构

### 模块1：任务信息卡
显示：
- 任务名称
- 模块名称
- 当前等级
- 任务编号

### 模块2：本次要求
例如：
- 使用哪几种原型 / 哪几个基础形
- 装饰或生成什么对象
- 构图或变形要求
- 附加要求

### 模块3：示范参考入口
可回看教学示范。

### 模块4：打印底稿入口
可生成打印底稿。

### 模块5：完成提醒
提示：
- 纸上完成后拍照上传
- 可先命名也可上传后命名

## 9.3 功能清单
- 查看任务内容
- 查看任务编号
- 回看示范
- 下载 / 打印底稿
- 跳转上传页
